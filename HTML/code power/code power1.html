코드의 힘

생활코딩바로가기 


      동영상 서비스의 동영상을
      우리 웹사이트에 포함시키는 방법

      youtube의 share 버튼
      옛날엔 source였음
      source? =code

      'Embed Video' 버튼
      태그로 이루어진 소스코드 등장
      태그중에 'iframe'이라는 태그

      iframe이란 태그가 뭔지 배우진 않았지만
      이게 HTML코드이고
      이 코드가 위에 있는 동영상을 의미한다
      라는 것은 알 수 있음
    

      이런 서비스를 통해
      한푼도 사용하지 않고
      동영상을 많은 사람에게 제공 가능

      이 엄청난 부품을
      iframe이란 코드 한줄로
      우리의 웹 페이지에 합성할 수 있게 되는 것.

      h1처럼 작은 부품
      또는 동영상과 같은 비싸고 거대한 부품을
      합성해서
      새로운 제품, 새로운 웹 페이지를 만들 수 있게 되는 것.

      이것이 "코드의 힘"
