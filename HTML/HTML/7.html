HTML이 중요한 이유

  블로그 같은곳에 글쓰기

  이런 편집기를 이용해서 글을 쓰는것은
  내부적으로 HTML 코드를 생산하는 행위였다.

  HTML을 몰라도
  HTML을 만들고 있었던 거임

  글씨 폰트사이즈 키우고 진하게 한 'coding'
  <h3> 태그로 둘러쌓인 'coding'
  어떤 차이?

  1
  <span style="font-size:24px">
    coding
  </span>

  
result :

  
coding
  

  2
  <h3>coding</h3>
  
result :

  
coding


  검색엔진은 누구의 편을 들까?
  1 처럼 겉만 제목처럼 꾸민 그냥 'coding' 문자
  2 처럼 제목태그를 붙인 real 제목 'coding' 문자

  당연히 2가 더 상위에 올라 오겠지?
  1은 한 100페이지 정도...검색엔진에 노출되지 않음
