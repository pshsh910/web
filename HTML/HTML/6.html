    HTML Paragraphs

    <p> </p> 중요
    HTML 'P'aragraphs
    The HTML <p> element defines a paragraph.
    A paragraph always starts on a new line, and browsers automatically add some white space
    (a margin) before and after a paragraph.
    어디서부터 어디까지 한 단락인지를 표시할 수 있기 때문에

    <hr>
    HTML 'H'orizontal 'R'ules
    The <hr> tag defines a thematic break in an HTML page,
    and is most often displayed as a horizontal rule.

    <br> 중요
    HTML Line 'Br'eak
    Use <br> if you want a line break (a new line) without starting a new paragraph
